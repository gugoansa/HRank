# La diferencia principal es el alcance y la reasignación: 

* var es de alcance de función y puede reasignarse; 
* let tiene alcance de bloque y puede reasignarse; 
* const tiene alcance de bloque y no puede ser reasignada ni re-declarada. 

No existen otros términos para declarar variables; estas tres son las únicas 
opciones en JavaScript. 


BLOQUE???
En JavaScript, un bloque es una sección de código encerrada entre llaves { }.
Por ejemplo:
{
  // Este es un bloque
  let x = 10;
  const y = 20;
  console.log(x + y);
}
